## 🔍 ANÁLISIS DE PROBLEMAS

```
Expected: {"email": "...", "password": "..."}
Actual: email=...&password=...
```
**Solución:** El endpoint de login necesita aceptar Form Data, no solo JSON.

### Problema 2: CSRF Token no funciona en logout
```
Error: {"detail":"Token CSRF invalido"}
```

**Solución:** Necesita obtener CSRF token antes de hacer POST.
